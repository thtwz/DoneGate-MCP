# DoneGate MCP v0.2 Bootstrap Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a bootstrap command that initializes a repository for DoneGate supervision and installs the bundled git hooks safely.

**Architecture:** Keep bootstrap logic in the service layer so the CLI and future MCP tooling can share it. Persist only minimal supervision state, and keep hook installation deterministic and explicit.

**Tech Stack:** Python 3.11+, argparse CLI, file-backed state, pytest.
