# DoneGate v0.3 Branch Context Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make active task context branch-aware and normalize repo-relative references against a stable repo root.

**Architecture:** Extend project and session state with repo-root and branch-binding metadata, then normalize path-bearing task fields at service boundaries. Keep the lifecycle rules untouched while making context lookup and path validation deterministic.

**Tech Stack:** Python 3.11+, argparse CLI, git branch detection, file-backed state, pytest.
